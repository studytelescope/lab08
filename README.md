Initialize Dockerfile
Set up ubuntu version

```sh
# base image
$ cat > Dockerfile <<EOF
FROM ubuntu:18.04
EOF
```

Add executable commands

```sh
# update initial packets and install compiler
$ cat >> Dockerfile <<EOF

RUN apt update
RUN apt install -yy gcc g++ cmake
EOF
```

Set word dir

```sh
# copy current dir to print/
$ cat >> Dockerfile <<EOF

COPY . print/
WORKDIR print
EOF
```
Set build Instructions

```sh
# make execute cmake targets
$ cat >> Dockerfile <<EOF

RUN cmake -H. -B_build -DCMAKE_BUILD_TYPE=Release -DCMAKE_INSTALL_PREFIX=_install
RUN cmake --build _build
RUN cmake --build _build --target install
EOF
```

Set log

```sh
# set new env variable
$ cat >> Dockerfile <<EOF

ENV LOG_PATH /home/logs/log.txt
EOF

# set a place of files storing
$ cat >> Dockerfile <<EOF

VOLUME /home/logs
EOF
```

Set up work dir

```sh
# set new work dir
$ cat >> Dockerfile <<EOF

WORKDIR _install/bin
EOF
```

Sst up point of entry

```sh
# set the point of entry
$ cat >> Dockerfile <<EOF

ENTRYPOINT ./demo
EOF
```
